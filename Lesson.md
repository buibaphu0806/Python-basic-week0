# Python Basic - Week 0

## Key Concepts

- Variables and Data Types  
- Input and Output  
- Conditional Statements (if)  
- Loops (for, while)  
- Functions  

## What I Understood Well

- A variable is a container used to store data values.
- Python is dynamically typed, so we do not need to declare the data type explicitly.
- Common data types include:
  - int (integer)
  - float (decimal number)
  - str (string)
  - bool (True/False)

- The `input()` function allows the user to enter data.
- The `print()` function displays output on the screen.
- The `if` statement is used to make decisions based on conditions.
- Loops (`for`, `while`) are used to repeat a block of code.
- A function is a reusable block of code that performs a specific task.
- The `return` keyword sends a value back to where the function was called.

## What Was Challenging

- Understanding when to use `print()` and when to use `return`.

- Controlling loops to avoid infinite loops.
